Tags Semânticas

Nesta aula, vamos começar a estudar as tags semânticas do HTML.

Este é um site criado com o objetivo de exemplificar a aplicação de tags semânticas e HTML básico a uma página da web, utilizando o IDE VSCode do GitHub CodeSpaces e a extensão Live Server.

Pesquisa de profissão

Coleta de dados relacionada à carreira de Front-end

Nesta aula, devemos realizar uma consulta de profissão da carreira de Front-end e coletar dados relacionados a salários, empregabilidade, riscos, etc.

Sistemas Operacionais

Funcionalidades básicas e acessórios

Vamos aprender os recursos básicos do sistema operacional Windows 8 e alguns dos seus acessórios.

Saúde e Segurança do Trabalho

Doenças Ocupacionais da Tecnologia

Quais são as doenças ocupacionais mais comuns que afligem os profissionais de tecnologia? Descobriremos também como evitá-las e garantir um ritmo e produtividade seguros, além de um guia básico de ergonomia no ambiente de trabalho do desenvolvedor Front-end.
